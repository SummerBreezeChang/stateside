# Submission Checklist

## Participation

- [ ] Confirm entrant/team eligibility in the official rules
- [ ] Join the hackathon on Devpost
- [ ] Choose exactly one track for this submission
- [ ] If entering as a team or organization, designate an authorized representative

## Build

- [ ] Project uses Codex and GPT-5.6
- [ ] Project is new or meaningfully extended during the submission period
- [ ] Dated commits/session evidence distinguish eligible work from any prior work
- [ ] Third-party software, data, media, and trademarks are licensed or authorized
- [ ] Hero flow runs consistently on the stated platform
- [ ] Included sample data is safe to publish and sufficient to test
- [ ] Usage/cost limits and failure states are handled

## Repository and testing

- [ ] Repository URL is ready
- [ ] Repository is public with relevant licensing, or shared with `testing@devpost.com` and `build-week-event@openai.com`
- [ ] README includes setup, sample data, configuration, and run instructions
- [ ] README explains how Codex accelerated the work and where human decisions were made
- [ ] README explains how GPT-5.6 and Codex contributed to the final result
- [ ] Working demo, test build, sandbox, or test account is free and accessible through judging
- [ ] Credentials and judge instructions are provided if access is private
- [ ] For a plugin/developer tool: installation, supported platforms, and no-rebuild test path are included

## Demo video

- [ ] Public YouTube URL
- [ ] Under three minutes
- [ ] Clear audio
- [ ] Shows the project actually working
- [ ] Explains what was built
- [ ] Explains how both Codex and GPT-5.6 were used
- [ ] No unlicensed music, copyrighted material, or third-party trademarks

## Devpost submission

- [ ] Category selected
- [ ] Project description explains features and functionality
- [ ] Repository URL added
- [ ] Demo video URL added
- [ ] `/feedback` Codex Session ID from the task where most core functionality was built added
- [ ] All English-language requirements satisfied
- [ ] Final submission completed before **Tuesday, July 21, 2026 at 5:00 PM PT**
