# Stateside

**Stateside** is the working title for an OpenAI Build Week 2026 project built with Codex and GPT-5.6.

Stateside helps international students compare potential first U.S. homes, understand how each fits their stay and daily life, distinguish verified facts from unknowns, and prepare next steps before committing.

## Current phase

**Experience defined; golden-path prototype next.** The research remains in [RESEARCH_INTERNATIONAL_STUDENT_HOUSING.md](./RESEARCH_INTERNATIONAL_STUDENT_HOUSING.md).
